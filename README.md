Documentção do site

Página de divulgação de padaria
Cadastro de Usuários
- Registro de Usuário: Um sistema de registro onde os proprietários de microempresas possam criar contas, fornecendo informações como nome, e-mail e senha.- Login: Opção para que os usuários já cadastrados possam acessar suas contas.
Formulário de Anúncio
- Informações da Empresa:- Campo para o nome da padaria.- Campo para descrever o diferencial da empresa (ex: produtos artesanais, ingredientes orgânicos, etc.).- Upload de Imagens:- descrição do cardápio.- Campo para upload de uma foto do estabelecimento.- Localização: Campo para inserir o endereço da padaria (sem integração com mapas, mas permitindo que o usuário forneça o endereço completo)
Exibição dos Anúncios
- Página de Listagem: Uma seção onde todos os anúncios de padarias são exibidos, com informações resumidas (nome, diferencial)
Página de Detalhes: Ao clicar em um anúncio, o usuário deve ser direcionado a uma página com detalhes completos.
- Busca: Funcionalidade de busca para que os usuários possam encontrar padarias por nome.
Avaliações e Comentários
- Sistema de Avaliação: Permitir que os clientes deixem avaliações e notas para as padarias.- Comentários: Opção para que os usuários comentem sobre suas experiências.1. Segurança e Privacidade- Proteção de Dados: Implementação de medidas de segurança para proteger as informações pessoais dos usuários o perguntas anônimas
Notificações
- Atualizações: Sistema de notificações para informar os usuários sobre novos comentários, avaliações ou atualizações em seus anúncios.
